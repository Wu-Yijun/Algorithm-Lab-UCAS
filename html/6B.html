跳舞机

Description

小A最近在练习跳舞毯。一个跳舞毯上有四个代表上下左右的按钮，屏幕上会依次展示四个方向的箭头。当某个方向的箭头出现的时候，小A必须用脚（左右脚都可以）踩一下相应的按钮。我们假定

  • 小A刚开始左脚在左箭头，右脚在右箭头上；
  • 脚踩一下箭头需耗费一个单位的能量；
  • 脚从一个箭头移动到另一个箭头需要额外消耗能量（见下表）；
  • 双脚不能同时踩在一个按钮上。
L R U D
L 0 3 2 4
R 1 0 3 2
U 3 2 0
D 1 3 2 0

第i行第j列表示从i箭头到j箭头移动所消耗的能量。

在给定跳舞毯的箭头序列后，请帮助小A计算如何跳舞才能够消耗最少的能量。

Input

第一行包含一个整数n (1<=n<=10)；接下来n行，每行包含一个由LRUD四个字母组成的字符串（长度在1-1000000之间），四个字母分别代表左右上下。

Output

对于每行，输出一个整数，表示所需要的最少能量

Samples

输入数据 1

1
RLUDLRDRD
Copy

输出数据 1

17
Copy

Hint

**样例解析：**

前两个字符RL：没有发生移动（消耗2个能量）；

接下来两个字符UD：右脚从R移动到U踩一下，接着移动到D踩一下（消耗7点能量）；

接下来两个字符LR：左脚踩一下L并移动到R（消耗5点能量）；

后续DRD三个字符：不需要移动（消耗3点能量）。
