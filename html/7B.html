Largest Number

Largest Number

Description

Suppose we have n n n integers. We want to combine them in a row so as to obtain a largest number. For example, consider three integers 13, 312, 343. We can combine them to get 13312343, 13343312, etc. The largest number we can obtain is 34331213.

Input

The input contains two lines. The first line contains one integer n n n ( 1 ≤ n ≤ 1000 1 \leq n \leq 1000 1≤n≤1000). The second line contains n n n integers and each integer is in the range [1,1000].

  • 50% test cases guarantee that n ≤ 10 n \leq 10 n≤10;
  • 20% test cases guarantee that 10 < n ≤ 100 10 < n \leq 100 10<n≤100;
  • 30% test cases have that 100 < n ≤ 1000 100 < n \leq 1000 100<n≤1000.

Output

The output contains one line which is the largest number we can obtain.

3
13 312 343
