使徒行者

使徒行者

Description

CIB 高级督察卓凯决定派卧底敢死队连续对最大的社团和兴进行攻击。每个卧底都有一个原始攻击值 a i a_i ai​ 和一个顽强值 b i b_i bi​。每个卧底发动攻击后都会暴露从而牺牲。英雄也会恐惧！一个卧底的恐惧值为其之前牺牲的卧底的顽强值总和。而一个卧底发动攻击时的真正攻击值为其原始攻击值与其恐惧值之差。为了给黑社会社团造成强大的震慑力：我们每个卧底英雄都不是盖的！卓 Sir 要求 IT 部的精英设计这队卧底的攻击顺序，使得真正攻击力最小的卧底的攻击力最大。

Input

输入数据的第一行为一个整数 n n n, 表示卧底的个数. 接下来 n n n 行每行两个正整数 a i a_i ai​ 和 b i b_i bi​，表示该名卧底的原始攻击值和顽强值。保证有

  • 对于 20% 的数据有 0 < n ≤ 10 0 < n \leq 10 0<n≤10；
  • 对于 40% 的数据有 0 < n ≤ 20 0 < n \leq 20 0<n≤20；
  • 对于 100% 的数据有 0 < n ≤ 1000 0 < n \leq 1000 0<n≤1000。
  • 对于全部的数据有 0 < a _ i , b _ i < 1 0 6 0 < a\_i, b\_i < 10^6 0<a_i,b_i<106。

Output

输出一个整数，为真实攻击力最小的卧底可以达到的最大攻击力。

Samples

输入数据 1

3
7 4
5 3
3 1
Copy

输出数据 1

3
Copy

Hint

最优攻击顺序为321：3先发起攻击，真实攻击值为3；接着2发起攻击，真实攻击为5-1=4；最后1发起攻击，真实攻击为7-1-3=3。因此最小攻击力为3。
