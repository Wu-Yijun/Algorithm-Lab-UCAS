Draw a circle

Draw a circle

Description

There are N N N points in a plane, each with a two-dimensional coordinate. We want to draw a circle which has at least two points within or on its border. You are asked to calculate the minimum radius of this circle.

Input

The first line is an integer N N N, denoting the number of points. Following are N N N lines. Each contains a pair of real number ( x , y ) (x, y) (x,y), denoting the coordinate of a point. ∣ x ∣ ≤ 1 0 9 , ∣ y ∣ ≤ 1 0 9 |x|\le 10^9,|y|\le 10^9 ∣x∣≤109,∣y∣≤109.

  • 40% of the test cases guarantee that N ≤ 1000 N\le 1000 N≤1000.
  • 70% of the test cases guarantee that N ≤ 25000 N\le 25000 N≤25000.
  • 90% of the test cases guarantee that N ≤ 100000 N\le 100000 N≤100000.
  • All test cases guarantee that N ≤ 200000 N\le 200000 N≤200000.

Output

Output only a real number rounding to 3 decimals after decimal point, denoting the minimum radius.
