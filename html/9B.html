Description

My birthday is coming up. Alas, I am getting old and would like to feel young again. Fortunately, I have come up with an excellent way of feeling younger: if I write my age as a number in an appropriately chosen base b b b, then it appears to be smaller. For instance, suppose my age in base 10 is 32. Written in base 16 it is only 20!

However, I cannot choose an arbitrary base when doing this. If my age written in base b b b contains digits other than 0 to 9, then it will be obvious that I am cheating, which defeats the purpose. In addition, if my age written in base b b b is too small then it would again be obvious that I am cheating.

Given my age y y y and a lower bound l l l on how small I want my age to appear, find the largest base b b b such that y y y written in base b b b contains only decimal digits, and is at least l l l when interpreted as a number in base 10.

Input

The input consists of a single line containing two base 10 integers y y y ( 10 ≤ y ≤ 1 0 18 10 \leq y \leq 10^{18} 10≤y≤1018 – yes, I am very old) and l l l ( 10 ≤ l ≤ y 10 \leq l \leq y 10≤l≤y).

Output

Display the largest base b b b as described above.
